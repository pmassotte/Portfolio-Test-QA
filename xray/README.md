# Xray/Jira – Campagne de tests API (Swagger Petstore)

## Objectif
Créer une campagne de tests complète dans **Jira** :
- Précondition
- Tests manuels (avec steps)
- Test Plan
- Test Execution (PASS/FAIL)
- Bugs liés aux tests

## Artefacts réalisés
- PRE-001 : Précondition d’accès à l’API
- TST-001 → TST-010 : 10 tests manuels (CRUD + recherche)
- TP-001 : Test Plan (regroupe tous les tests)
- TE-001 : Test Execution (résultats PASS/FAIL)
- BUG-001 / BUG-002 : anomalies créées depuis l’exécution et liées aux tests
